### Enoncé

Nous avons trouvé un système étrange qui sert à la communication pour l'ennemie
Un système dont le nom est le même qu'un célèbre guerrier gaulois. Nous avons besoin
de ton aide pour pouvoir enregistré un de nos équipement pour pouvoir intéragir avec leur
système de communication.
Ton objectif : T'enregistrer sur le serveur de communication ennemi avec le softphone de ton choix.
Pour récupérer l'information dont nous avons besoin, appelle le numéro 666666

Format attendu du flag : `FMCTF{}` (mots et/ou nombres séparés par des underscores : **_** et les mots sont en minuscule)

### Ressource
Les informations nécessaires à votre connexion au serveur de communication se trouve dans votre channel discord

### Précisions : 
- Pour ce chall, vous aurez besoin d'un softphone (que vous soyez en présentiel ou en distanciel).
- Windows : Zoiper ou MicroSIP; Linux : Zoiper ou Twinkle; Mac : Zoiper (Juste des propositions, si vous en avez d'autres,utilisez les, pas de problèmes).
- La téléphonie a du mal avec le NAT donc soit vous mettez votre Softphone sur une VM en accès par pont (uniqument possiblepour les personnes en présentielles) ou vous avez votre softphone sur votre machine physique (possible pour les personnes enprésentiel et en distanciel).
- Le son est utile pour les challs suivants de téléphonie, il sera sans doute de meilleure qualité sur votre machine physiqueque sur une VM.
- Les informations de connexion (numéro attribué et mot de passe) ont été envoyées sur Discord
- Vous pouvez être plusieurs enregistrés avec le même numéro et le même mot de passe (chaque groupe a un compte et chaquepersonne du groupe peut s'enregistrer, si vous avez envie)
